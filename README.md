# Návrh relační databáze fotbalového informačního systému
Dobrý den,

jmenuji se Petr Ševčík a potřebuji navrhnout jednoduchou databázi pro evidenci fotbalových klubů, hráčů a zápasů. Momentálně si údaje ukládám do různých poznámek a tabulek a stává se to nepřehledné. Rád bych měl všechno na jednom místě a bez zbytečného opakování dat.


Databáze by mi měla umožnit:

  -organizovat informace o klubech, jejich městech a stadionech,
    
  -evidovat hráče a jejich zařazení do kategorií,
  
  -ukládat soutěže a typy soutěží,
  
  -vést záznamy o jednotlivých zápasech, výsledcích, stadionech a čase konání,
  
  -propojit tyto údaje tak, aby se daly snadno vyhledávat.


Co potřebuji evidovat:

  -města a adresy stadionů,
  
  -fotbalové kluby,
  
  -hráče,
  
   -věkové a výkonnostní kategorie,
    
   -soutěže,
  
   -zápasy a jejich výsledky.
  

Rád bych, aby databáze byla přehledná, normalizovaná, odstranila duplicitní data a dala se v budoucnu využít jako základ pro jednoduchou webovou aplikaci.

Děkuji.
